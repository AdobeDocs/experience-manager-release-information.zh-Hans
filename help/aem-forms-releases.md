# AEM [!DNL Forms]发行版{#aem-forms-releases}

Adobe Experience Manager[!DNL Forms]是部署到AEM上的应用程序。 它将引人入胜的Web和移动体验扩展到企业表单和文档，使组织能够完成复杂的数字交互，同时改进用户体验和扩大业务范围。 AEM快速开始中附带的现成AEM [!DNL Forms]具有有限的功能集。 AEM [!DNL Forms] add-on包提供高级功能，如与[!DNL Adobe Analytics]、[!DNL Adobe Sign]、文档服务、工作流引擎等集成。

>[!NOTE]
>
>AEM Forms在计划的AEM Service Pack和Cumulative Fix Pack发布日期后一周发布加载项包。

以下列表包含所有发布到今天的AEM [!DNL Forms]加载项包、相应的AEM版本（先决条件）、包的下载链接和其他有用信息：

## 要记住的事{#things-to-remember}

* 上面列出的AEM Forms附加程序包适用于OSGi上的AEM [!DNL Forms]和JEE上的AEM [!DNL Forms]。
* 所有AEM [!DNL Forms]加载项包和快速修复都是累积的；即，加载项包和快速修复包含相应版本组件的所有前面的加载项和快速修复。 例如，如果您正在安装快速修复2.4.50，它已经包括AEM [!DNL Forms] add-on package 2.4.26。

   >[!NOTE]
   >
   >累积附加包和快速修复意味着前两个版本号相同。

* 请始终安装最新可用的Service Pack/Feature Pack加载项。 强烈建议使用此插件，因为它包括安全性、性能、稳定性，以及自基本版本的通用性发布以来发布的重要客户修复和增强功能。
* 无论您何时安装Service Pack或从先前版本的AEM升级到更新版本，请始终安装相应的AEM [!DNL Forms]加载项包。 例如，如果从AEM 6.1 [!DNL Forms]加载项包版本2.2.4升级到AEM 6.1 Service Pack 1，则在安装AEM 6.1 Service Pack 1后，请安装AEM [!DNL Forms]加载项包版本2.4.50。
* 不支持卸载AEM [!DNL Forms]加载项包。
